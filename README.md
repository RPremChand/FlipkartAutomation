# Flipkart Automation

## Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/RPremChand/FlipkartAutomation.git
   cd FlipkartAutomation
   ```
2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Set up environment variables:**
   Create a `.env` file and add the necessary configuration settings. 

## Usage Guide

To run the automation scripts:
```bash
python main.py
```

Replace `main.py` with the appropriate script you want to execute. 

For additional options, run the script with `--help`.
